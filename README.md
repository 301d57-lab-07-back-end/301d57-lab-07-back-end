# lab-07-back-end

**Authors**: Matt Stuhring, Chris 

## Overview
Provide the user with weather data for the location they enter.

## Getting Started
- Clone repository
- create .env file and set the port number
- From root directory, npm install
- node server.js
- Go to localhost at your set port number

## Architecture
- express
- cors
- dotenv
- NodeJS
- HTML/CSS
